## 🚀 Quick Start Guide

### 1. 📥 **Grab the Code**

```bash
git clone <repository-url>
cd "Cohort Hackathon"
```

### 2. 📦 **Install the Magic**

```bash
npm install
```

### 3. 🔥 **Fire It Up**

```bash
npm run dev
```

### 4. 🌐 **Experience the Magic**

Open your browser and visit: `http://localhost:5173`

> 🎉 **Boom!** You're now running COCOFIT locally. Time to explore!
